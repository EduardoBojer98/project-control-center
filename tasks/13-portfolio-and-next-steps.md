# 13 - Portfolio And Next Steps

## Goal

Turn your learning into professional growth.

## Learn

- how to present projects
- how to write a strong README
- how to explain architecture decisions
- how to show progress on GitHub
- how to keep improving after the first full-stack app

## Tasks

- write a clean README for your final project
- add screenshots
- explain the stack you used
- explain what you learned
- list future improvements
- publish the repo
- deploy the app and add the live link

## Next technologies to explore after this path

- React Query
- Zod
- Prisma
- PostgreSQL
- WebSockets
- Docker
- CI/CD
- Next.js

## Done when

- your work is easy for others to understand
- you can talk about your app like a developer, not just a student
- you know what to learn next without feeling lost
