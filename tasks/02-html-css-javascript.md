# 02 - HTML CSS JavaScript Foundations

## Goal

Build strong frontend basics before React.

## Learn

- semantic HTML
- CSS layout with flexbox and grid
- responsive design
- JavaScript variables, functions, arrays, objects
- loops and array methods
- DOM events
- fetch basics

## Tasks

- build a static landing page with HTML and CSS
- make it responsive for mobile and desktop
- create a JavaScript to-do list without React
- use `map`, `filter`, and `find`
- fetch data from a public API and show it in the page

## Mini exercises

- build a navbar, card, and form with plain HTML/CSS
- create a button click counter in JavaScript
- fetch users from an API and render them in the DOM

## Done when

- you can build a small page without a framework
- you understand how the browser runs JavaScript
- you can explain what `fetch` returns
