# 09 - Authentication And Security

## Goal

Learn how users sign in and how apps protect data.

## Learn

- authentication vs authorization
- sessions vs tokens
- password hashing
- protected routes
- backend auth middleware
- role-based access basics
- secure environment variables

## Tasks

- create a users table
- build register and login endpoints
- hash passwords safely
- store the logged-in user securely
- protect dashboard routes
- prevent unauthorized users from accessing private data

## Mini exercises

- create a logout route
- add a current user endpoint
- hide certain UI for non-admin users

## Done when

- you can explain how login works end to end
- you know why plain text passwords are never acceptable
- your app can protect private routes and data
