# 07 - Backend With Node And Express

## Goal

Learn how to build a backend server for your React app.

## Learn

- what Node.js does
- Express basics
- routes
- controllers
- request and response
- JSON APIs
- middleware basics
- environment variables

## Tasks

- create a `server/` folder
- initialize a backend project
- build a small Express server
- create a `/api/health` route
- create a `/api/projects` route
- return JSON data
- separate routes from controller logic

## Mini exercises

- create a `/api/tasks` route
- log every request with simple middleware
- add a route that returns a single project by id

## Done when

- you understand how the frontend talks to the backend
- you can create REST endpoints by yourself
- you know what middleware is doing
