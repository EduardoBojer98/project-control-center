# 03 - TypeScript Basics

## Goal

Learn how TypeScript helps you write safer frontend and backend code.

## Learn

- primitive types
- arrays and objects
- function types
- interfaces and type aliases
- unions
- optional properties
- generics
- type narrowing

## Tasks

- rewrite JavaScript examples into TypeScript
- create types for a user, project, and task
- write functions with typed parameters and return values
- create a reusable generic list type
- practice narrowing union types with condition checks

## Mini exercises

- type a login form object
- type an array of projects
- create a function that formats task status labels

## Done when

- you stop guessing data shapes
- you can read TypeScript errors without fear
- you understand why shared types matter in full-stack apps
