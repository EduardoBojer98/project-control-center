# 10 - Auth Security And Real App Concerns

## Goal

Learn the pieces that make a project feel like a real application.

## Learn

- authentication vs authorization
- sessions vs JWT at a high level
- password hashing
- protected API routes
- current user patterns
- role-based access basics
- environment variable safety
- common security mistakes

## Tasks

- build register, login, logout, and current-user endpoints
- hash passwords correctly
- protect project and task routes
- connect frontend auth state to backend auth responses
- hide or block unauthorized actions
- handle unauthorized and expired sessions gracefully

## Mini exercises

- add admin-only behavior
- return a clean unauthorized response shape
- add frontend route protection for private pages

## Done when

- you can explain auth end to end
- you understand what your app must protect
- your app starts behaving like production software
