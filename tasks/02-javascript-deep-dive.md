# 02 - JavaScript Deep Dive

## Goal

Level up JavaScript far beyond the basics so React and backend code feel natural.

## Learn

- scope and closures
- execution context
- this keyword
- array methods deeply
- objects and reference vs value
- destructuring
- spread and rest
- modules
- asynchronous JavaScript
- promises
- async and await
- error handling
- map, filter, reduce in real code
- event loop basics

## Tasks

- explain closures with your own example
- create reusable utility functions with `map`, `filter`, and `reduce`
- rewrite callback-based logic using promises
- build a small async data loader with loading and error states
- practice object and array transformations
- learn how reference mutation causes bugs
- explain the event loop in simple words

## Mini exercises

- group a list of tasks by status using `reduce`
- create a debounce function
- write a function that retries a failed async request
- transform API data into UI-friendly data

## Done when

- JavaScript feels like a tool, not a wall
- async code makes sense to you
- you can read real production JavaScript without getting lost
