# 06 - Routing Forms And Validation

## Goal

Build real app structure and user input flows.

## Learn

- React Router
- nested layouts
- route-level thinking
- controlled forms
- validation strategies
- touched and error state
- form UX
- optional intro to React Hook Form and Zod later

## Tasks

- create routes for dashboard, projects, tasks, login, and settings
- create shared layout components
- build a login form and a project creation form
- validate fields manually first
- then compare that to a library-based approach later
- create a not found route
- handle success and error feedback cleanly

## Mini exercises

- build a multi-field form with validation messages
- disable submit during invalid state
- create reusable field and error components

## Done when

- you can structure multi-page React apps well
- your forms are predictable and maintainable
- validation no longer feels messy
