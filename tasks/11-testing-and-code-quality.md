# 11 - Testing And Code Quality

## Goal

Learn how to trust your code and refactor safely.

## Learn

- unit tests
- integration tests
- component tests
- API tests
- linting mindset
- testable architecture

## Tasks

- test utility functions
- test React components with user interaction
- test a form validation flow
- test backend endpoints
- test success, failure, and edge cases
- identify code that is hard to test and refactor it

## Mini exercises

- test a custom hook
- test a reducer or formatter
- test unauthorized API access

## Done when

- you know what to test and why
- refactoring feels safer
- bugs become easier to isolate
