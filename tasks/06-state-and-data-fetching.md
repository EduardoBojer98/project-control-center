# 06 - State And Data Fetching

## Goal

Learn how frontend apps work with data.

## Learn

- local state
- derived state
- loading and error states
- fetch requests
- async and await
- separating UI from data logic
- optional introduction to React Query later

## Tasks

- fetch a list of projects from a public or fake API
- show loading, error, and empty states
- create a file for API functions
- keep page components focused on rendering
- filter and sort fetched data
- add a create form that updates local state first

## Mini exercises

- fetch posts and display them
- create a reusable loading component
- move fetch logic into a separate file

## Done when

- you know where fetch logic should live
- you can handle asynchronous UI states well
- your components are not mixing too many responsibilities
