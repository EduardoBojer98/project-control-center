# 08 - Node Express And API Design

## Goal

Learn backend fundamentals in a way that connects directly to your React work.

## Learn

- Node.js basics
- Express setup
- routing
- controllers
- services
- middleware
- request validation basics
- REST API design
- status codes
- error handling

## Tasks

- create a `server/` project with TypeScript
- create health, projects, tasks, and auth routes
- separate route handlers from business logic
- return proper status codes
- create error-handling middleware
- validate incoming request data
- design endpoints that your React app can consume clearly

## Mini exercises

- create CRUD routes for projects
- create a route for tasks by project id
- return validation errors in a consistent shape

## Done when

- backend code feels understandable
- you can create APIs intentionally
- your frontend and backend start speaking the same language
