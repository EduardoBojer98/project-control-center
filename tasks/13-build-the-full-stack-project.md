# 13 - Build The Full-Stack Project

## Goal

Pull everything together into one serious learning project.

## Build this app

Build a project management application with:

- register and login
- dashboard
- project CRUD
- task CRUD
- task status updates
- comments
- user settings
- protected routes

## Suggested stack

- React
- TypeScript
- React Router
- Tailwind CSS
- Node.js
- Express
- PostgreSQL
- Prisma

## Suggested build order

1. scaffold the client and server
2. create shared types and data shapes
3. build frontend routes and layouts
4. build backend routes and controllers
5. connect PostgreSQL with Prisma
6. connect frontend to backend
7. add auth
8. add tests
9. deploy it

## Rules for yourself

- keep code modular
- avoid giant page components
- avoid giant controller files
- type your data carefully
- make loading and error states real
- refactor as you learn

## Final checklist

- app has a clear folder structure
- frontend is typed
- backend is typed
- database schema is clean
- auth works
- CRUD works
- testing exists
- app is deployed

## Done when

- you can explain the architecture without guessing
- you can demo the entire app confidently
- you feel capable of starting the next app with less guidance
