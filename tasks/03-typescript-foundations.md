# 03 - TypeScript Foundations

## Goal

Use TypeScript with confidence instead of treating it like annoying syntax.

## Learn

- type aliases and interfaces
- unions and intersections
- optional and nullable fields
- function typing
- arrays and records
- generics
- utility types
- narrowing
- inferred types
- when to use explicit types

## Tasks

- type users, projects, tasks, and auth payloads
- type a fetch function response
- create reusable generic helper types
- use `Record`, `Pick`, `Omit`, and `Partial`
- create union types for task status and user roles
- practice safe handling of nullable API data

## Mini exercises

- type a project card prop object
- type a form state object and error object
- create a generic API response type
- model loading, success, and error states with a union

## Done when

- you understand the types you write
- you can model app data cleanly
- TypeScript starts helping your thinking, not slowing it down
