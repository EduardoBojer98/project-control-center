# 12 - Build The Full Project

## Goal

Bring everything together into one full-stack application.

## Build this app

A project management app with:

- register and login
- dashboard
- create project
- edit project
- create task
- update task status
- comments
- user settings

## Suggested build order

1. create frontend routes and layouts
2. create shared types
3. create backend routes
4. connect database models
5. connect frontend to backend
6. add auth
7. add testing
8. deploy it

## Rules for yourself

- keep commits small
- name files clearly
- extract reusable components
- type your data
- separate UI, data, and backend logic
- do not jump ahead when basics are still weak

## Final checklist

- frontend works
- backend works
- database works
- auth works
- app handles loading and errors
- project is deployed
- you can explain every folder in your codebase

## Done when

- you can demo the app from start to finish
- you understand the architecture
- you feel ready to start a second project with less guidance
