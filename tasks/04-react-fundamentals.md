# 04 - React Fundamentals

## Goal

Learn how to build UI with components.

## Learn

- JSX
- components
- props
- state
- events
- conditional rendering
- list rendering
- lifting state up
- `useEffect`

## Tasks

- create a new React app from scratch
- build a page using reusable components
- create a counter component
- create a task list component
- pass typed props to child components
- handle input changes with state
- show loading, empty, and success states

## Mini exercises

- build a reusable button component
- build a task card component
- build a search input that filters a list

## Done when

- you can split UI into components naturally
- you understand when state should live in parent components
- you can render dynamic data with confidence
