# 04 - Modern React Fundamentals

## Goal

Build a strong React mental model on top of your frontend experience.

## Learn

- component composition
- props
- local state
- event handling
- conditional rendering
- list rendering
- derived UI
- lifting state
- effect basics
- reusable component patterns

## Tasks

- build a small React app from scratch with Vite
- split one screen into reusable components
- create a task list with filtering
- create controlled form fields
- manage local state correctly
- identify what should be state and what should be derived
- practice parent-child data flow

## Mini exercises

- build a modal component
- build tabs with local state
- create an empty state and loading state component

## Done when

- you can think in components naturally
- you understand how React re-renders work at a high level
- you stop putting all logic in one file
