# 11 - Deployment And DevOps Basics

## Goal

Learn how to put your app online.

## Learn

- environment variables in production
- build process
- frontend deployment
- backend deployment
- database hosting
- domains and HTTPS
- logging basics

## Tasks

- deploy the frontend
- deploy the backend
- connect the frontend to the live backend
- connect the backend to a hosted database
- add environment variables safely
- verify the production app works

## Mini exercises

- create a production checklist
- test the deployed API with real data
- write down common deployment mistakes

## Done when

- your project is live
- you can explain your deployment setup
- you understand the difference between local and production environments
