# 10 - Testing

## Goal

Learn how to verify your app instead of guessing.

## Learn

- unit testing
- integration testing
- component testing
- API testing
- manual testing mindset

## Tasks

- test a utility function
- test a React component
- test a form validation rule
- test a backend route
- test a success case and a failure case

## Mini exercises

- write a test for a button click
- test a function that formats dates or statuses
- test a protected route response

## Done when

- you understand what should be tested
- you can catch regressions with confidence
- your project feels safer to change
