# 12 - Deployment And Production Thinking

## Goal

Learn how to move from local development to a working production app.

## Learn

- production builds
- environment variables
- frontend deployment
- backend deployment
- managed databases
- logging basics
- debugging live issues
- production tradeoffs

## Tasks

- deploy the React frontend
- deploy the Node backend
- connect both environments correctly
- connect a hosted PostgreSQL database
- configure environment variables safely
- test login and CRUD in production
- create a release checklist

## Mini exercises

- document local vs production env values
- test one broken deploy scenario and fix it
- add basic request logging

## Done when

- your app is live
- you understand how all pieces connect in production
- deployment stops feeling like a black box
