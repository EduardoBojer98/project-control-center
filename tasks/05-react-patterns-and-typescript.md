# 05 - React Patterns With TypeScript

## Goal

Combine React and TypeScript the way real projects need them.

## Learn

- typed props
- typed hooks
- typed event handlers
- typed forms
- reusable component APIs
- children and composition
- custom hooks
- separating UI and logic

## Tasks

- type all component props clearly
- create a reusable button and input component
- build a custom hook for toggling or filtering data
- type `onChange`, `onSubmit`, and click handlers correctly
- separate page logic from presentational components
- create a typed status badge system

## Mini exercises

- create a typed modal component
- create a typed table or list component
- create a `useDebounce` hook with types

## Done when

- your React types are clean and readable
- your components feel reusable and intentional
- you can design component APIs with confidence
