# 05 - React Router And Forms

## Goal

Learn page structure and user input handling.

## Learn

- client-side routing
- nested routes
- route layouts
- form state
- controlled inputs
- validation basics
- error messages

## Tasks

- add React Router to your app
- create pages for home, login, dashboard, and projects
- create a shared layout
- build a login form with controlled inputs
- show validation messages for invalid input
- add a not found page

## Mini exercises

- create a sidebar layout
- validate email and password length
- redirect users after a fake login

## Done when

- you can set up routes without confusion
- you understand controlled forms
- you can build a clean app layout
