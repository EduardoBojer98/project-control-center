# 01 - Tools And Workflow For Modern Projects

## Goal

Sharpen the workflow skills needed for React and TypeScript projects.

## Focus

This is not a beginner terminal lesson.
This is about working faster and more confidently in real app projects.

## Learn

- package.json scripts
- npm workflows
- Vite basics
- ESLint and formatting
- environment variables
- Git branches and clean commit habits
- reading stack traces and build errors
- project structure decisions

## Tasks

- explain every field in a typical `package.json`
- create scripts for dev, build, lint, and preview
- learn how Vite bootstraps a React app
- practice reading TypeScript and runtime errors
- create a branch workflow for feature work
- write better commit messages
- understand `.env` files and what should never be committed

## Mini exercises

- break an import path and fix the error
- break a TypeScript type and explain the compiler error
- compare a runtime error vs a compile-time error

## Done when

- you can debug toolchain problems without panic
- you understand what your scripts are doing
- your project setup feels intentional, not magical
