# 01 - Tools And Workflow

## Goal

Learn the tools every full-stack developer uses daily.

## Learn

- terminal basics
- folders and files
- VS Code basics
- Git basics
- GitHub basics
- npm and package managers
- how to read error messages

## Tasks

- create, rename, move, and delete files from the terminal
- learn `pwd`, `ls`, `cd`, `mkdir`, `rm`, `touch`
- create a Git repo and make commits
- create branches and merge a branch
- push code to GitHub
- install a package and understand where it goes
- read a `package.json` file and explain what scripts are

## Mini exercises

- create a test folder and practice terminal commands
- make 3 small commits with clear commit messages
- break something on purpose and read the error

## Done when

- you are comfortable in the terminal
- Git does not feel mysterious
- you understand what `npm install` and `npm run` do
